# OOP Exercises (Java)

Repository containing Object-Oriented Programming exercises completed as part of university coursework.

Each folder represents a separate project focused on practicing Java OOP concepts and JavaFX applications.

JavaFX application that calculates roots of a quadratic function based on coefficients a, b, c.
